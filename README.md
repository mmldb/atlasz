# Digitális Kárpát-medencei Néprajzi Atlasz

Forrásalapú, interaktív néprajzi atlasz pilot.

## MVP

- Vite + React + Leaflet térkép
- 3 pilot tájegység: Palócföld, Sárköz, Kalotaszeg
- Layer UI: építészet, női viselet, férfi viselet
- Forráshoz kötött adatpanel
- Review státuszok és approve UI placeholder

## Lokális futtatás

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
